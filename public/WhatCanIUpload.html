What can I upload?

Images

You can upload images upto 50MB.

Supported file types include: png, jpg (and all variants), webp, gif and more. We do not accept .RAW images are they are too big ;P

Files are stored forever unless we deem it "inappropriate", which isn't much.

If you don't like this then sorry you can use other services such as ghostfiles (They are a good file hosting service however they have stricter rules regarding images)


Files

Unlike images, files can be up to 80MB in size.

Literally every single file type is supported for the file upload.

Files are stored forever however if you lack an account and the file is 51MB+ deletion for such file will be considered after 3 months. you can contact us here for exemptions.
